# Pneumonia Detection Using Convolutional Neural Networks (CNN)

## Overview:
This project implements a Convolutional Neural Network (CNN) to classify chest X-ray images as Normal or Pneumonia. The model is built using TensorFlow/Keras and trained on a labeled chest X-ray dataset.

The project demonstrates the complete workflow of building a CNN from scratch, including image preprocessing, model training, saving the trained model, and predicting the class of a new chest X-ray image.

## Dataset:
The dataset used in this project is sourced from Kaggle

**Link:** https://www.kaggle.com/datasets/mitgandhi10/dataset-for-cnn/code
